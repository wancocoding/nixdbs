Linux Development Bootstrap
===


在Linux上快速搭建一个合适的开发环境,如数据库,缓存,编辑器,特定语言开发,帮助文档,开发工具等等



## 特性

* 可以选择国内线路或者国外线路安装
* 无人值守
* 支持的操作系统有`OSX`, `Debian/Ubuntu`, `Archlinux`, `Fedora/RedHat`, `OpenSuse`
* 多种选择,支持最小化(只安装非常必须的语言工具)
* 可以定制特定语言所需环境和工具
* 支持导入自己的dotfiles
* 大量常用的操作简化,比如git新建仓库
